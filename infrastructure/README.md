# Nyx Global Infrastructure

This folder contains the global ecosystem platform services:

- `registry-server/`: package registry + auth + verification + search + docs endpoints
- `collab-server/`: global developer collaboration + knowledge + analytics service
- `mirror-agent/`: distributed mirror synchronization agent
- `build-worker/`: deterministic multi-target build verification worker
- `api/openapi.yaml`: public API contract
- `api/collab_openapi.yaml`: collaboration API contract
- `deploy/`: docker-compose and k8s examples
- `docs/`: architecture, database design, deployment guides
